# analise_gasolina_petrobras

Este projeto tem como objetivo desenvolver um modelo de Machine Learning capaz de prever o comportamento futuro de ações no mercado financeiro com base em dados históricos e indicadores técnicos. A proposta busca identificar padrões nos movimentos de preços e auxiliar investidores na tomada de decisões mais informadas sobre compra ou venda de ativos.

Os mercados financeiros são altamente dinâmicos e influenciados por diversos fatores econômicos, políticos e comportamentais. Bolsas como a B3 no Brasil e a New York Stock Exchange movimentam trilhões de dólares diariamente. Com o avanço da ciência de dados e da inteligência artificial, técnicas de Machine Learning vêm sendo aplicadas para analisar grandes volumes de dados históricos e detectar tendências que podem indicar movimentos futuros do mercado.
